# Admin Role

Administrators have full system access and manage the overall platform.

## Key Responsibilities

1. **User management** (create/edit/delete users)
2. **Role assignment** (strategist, compliance, admin)
3. **System configuration**
4. **Payment settings** (Stripe, Coinbase)
5. **Email templates**
6. **Integration management** (DocuSign, etc.)

## Permissions

Admins have full access to:
- All users and roles
- System settings
- Payment configuration
- Integration settings
- Audit logs
- All data across the platform

## Routes

Admin routes would typically include:
- `/admin/users` - User management
- `/admin/settings` - System settings
- `/admin/integrations` - Third-party integrations
- `/admin/billing` - Platform billing settings
- `/admin/logs` - Audit logs
