# Ariex Documentation

This folder contains documentation for the Ariex Tax Strategy Platform, organized by user roles and their points of view.

## User Roles

| Role | Description | Folder |
|------|-------------|--------|
| **Strategist** | Tax strategists who manage clients and create tax strategies | [strategist/](./strategist/) |
| **Client** | End users who receive tax strategy services | [client/](./client/) |
| **Compliance** | Internal team managing compliance and oversight | [compliance/](./compliance/) |
| **Admin** | System administrators | [admin/](./admin/) |

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Phosphor Icons
- **State**: Zustand
- **API**: oRPC
- **Database**: PostgreSQL (TypeORM)
- **Payments**: Stripe, Coinbase
- **Signatures**: DocuSign / SignatureAPI
